姿势：
	swtich(表达式)   // 大括号是 swtich 自带的
	{
		case 选项：
				很多代码；
				break;
		case 选项：
				很多代码；
				break;
		case 选项：
				很多代码；
				break;
		case 选项：
				很多代码；
		#defalut（默认的）
					很多代码；
					break;
	}
解释：
		选项： 必须是整型（字符），并且不能重复
		表达式的值是多少，就会选择相对于的 选项 从而取执行其对应的 代码
		default 可以省略不写
#break和switch结合，立马结束switch语句
